# Root Holding Area

This directory holds loose root-level documentation that was removed from the governance repo root during consolidation on 2026-03-06.

These files were preserved rather than deleted because they may still contain useful planning, prompt, or specification material, but they are not part of the intentional active root documentation surface.

Promotion criteria:

- move to `docs/canonical/` only if the file is confirmed canonical doctrine
- move to `docs/plans/active/` only if the file is confirmed current and owned
- move to another governed location only when ownership and status are explicit
- otherwise keep here or archive further with an explicit report entry
