# Ecosystem Plans

This directory is the governed home for cross-repository implementation plans,
roadmaps, plan sets, and completed planning packs.

## Layout

- `active/` - current plans that still guide near-term implementation.
- `archive/` - superseded, completed, dated, or historical plan material.

Root-level plan sets should not live at the ecosystem repo root. Move them here
unless they are service-specific, in which case they belong inside the owning
service repo.

## Archived Plan Sets

| Directory | Contents |
|-----------|----------|
| `archive/dataforge-local-analytics-2026-04-17/` | ForgeCommand to DataForge Local local-systems analytics plan set |

| `archive/doctrine/` | Doctrine implementation references |
